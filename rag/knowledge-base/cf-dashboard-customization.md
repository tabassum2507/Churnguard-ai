---
title: "Customizing Your FlowMetric Dashboard"
category: config_fixes
---

Getting your FlowMetric dashboard set up to match how you actually work makes a noticeable difference in day-to-day speed. To start editing, click the Edit Dashboard button in the top right of any dashboard view. From there you can drag widgets to reorder them, resize them by pulling the corners, and remove anything you never look at by clicking the X that appears in edit mode.

The widget library has more than twenty options, including milestone timelines, budget burn gauges, team workload charts, and activity feeds. If you manage multiple projects, the cross-project rollup widget is the most valuable one to add — it gives you a single-card status indicator for every active project at once.

You can create multiple dashboard tabs for different purposes. A common setup is one tab for project health and a second for team utilization, so you can switch context quickly without scrolling past irrelevant widgets.

Dashboard layouts are personal by default, meaning your changes don't affect what your teammates see. Workspace admins can push a standard layout to all new users from the workspace settings, which is useful for keeping a baseline consistent across large teams while still letting individuals customize from there.
